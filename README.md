# web-angularjs-javascript-yeoman

# Recipe

FROM [codenvy/node](https://hub.docker.com/r/codenvy/node/)

# Commands

Command #1
| Command  |
| :------------- |
| `cd ${current.project.path}` |
| `npm install && bower install` |
| `gulp serve` |
